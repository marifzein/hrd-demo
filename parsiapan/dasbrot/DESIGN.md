# Design System Documentation

## Theme Overview

This document outlines the core visual theme for our product, derived from the design tokens.

### Color Palette

Our interface utilizes a **light** color mode, providing a bright and airy experience. The primary brand color is `#5175c3`, a strong, dependable hue. It is complemented by a secondary color, `#ffe200`, offering a sophisticated contrast, and an accent tertiary color, `#7b0000`, for highlights. Neutral elements and backgrounds are based on `#777777`.

### Typography

*   **Headlines** use the `Manrope` font, chosen for its modern and geometric aesthetic, ensuring readability and impact.
*   **Body text** is set in `Inter`, a highly versatile and legible typeface, optimized for UI.
*   **Labels** also utilize `Inter` for consistency and clarity across interactive elements.

### Shape and Form

Our UI elements feature a **subtle roundedness (1)**, offering a friendly yet polished appearance, avoiding overly sharp or pill-shaped corners.

### Spacing

The design adheres to a **normal (2)** spacing rhythm, balancing density and visual breathing room, leading to a comfortable user experience.
